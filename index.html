EngageOne Communicate Video API Demo

Here is a description of this page.

Step 1 - Click the "Generate Token" button to generate an authorization token to be used in the next steps.

Step 2 - Click the "Generate URL" button to generate the generic video URL.

Step 3 - Click the "Get Data Source ID" button to be used in the next steps.

Step 4 - Click the "Get Record IDs" button to be used to get the list of PURLS in the next step.

Step 5 - Click the "Append Record IDs to URL" button to be used to generate the finalized list of PURLS!
